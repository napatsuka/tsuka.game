# 弾幕回避ゲーム (Bullet Hell)

ローカルで開いて遊べるシンプルな弾幕回避ゲームです。

特徴:
- タイトル画面からスタートボタンで始まる
- 10秒ごとにレベルが上がり弾幕が激しくなる
- 弾に当たるとゲームオーバー
- スワイプ/タッチで自機を移動可能（モバイル対応）

使い方:
1. `index.html` をブラウザで開く
2. スタートボタンを押してゲーム開始
3. タッチ/ドラッグで自機を移動して弾を避ける

開発メモ:
- ゲームループは `requestAnimationFrame` を使用
- 弾は複数の軌道（直線・正弦波・螺旋・ホーミング）を持つ
- レベルは10秒毎に上昇し、弾の頻度・種類が増える

拡張案:
- スコア表示、ボス波、当たり判定の調整
- エフェクト・サウンドの追加
- 難易度設定・ハイスコア保存

Enjoy!

---

画像を差し替える
- 自機画像: `assets/images/player.png` を置くと自機が PNG で表示されます（推奨サイズ: 64x64〜128x128、透過推奨）。
- タイトル画像: `assets/images/title.png` を置くとタイトル画面に表示されます（推奨サイズ: 横幅 400px 程度）。

画像がない場合は既存の円表示がフォールバックで使われます。
