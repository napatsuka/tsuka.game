# Handy Tetris

簡単に遊べる Tetris デモ。

操作
- 左 / 右: 移動
- 下: ソフトドロップ
- ↑: 回転
- Space: ハードドロップ

- P: 一時停止

実行
- ブラウザで `c:\workspace\tetris\index.html` を開く

備考
- ハイスコアは localStorage に保存されます。
- タイトル用の画像を `assets/title.png` に配置すると、タイトル画面に表示されます。推奨サイズは幅 800px 前後の PNG（透過可）です。
- さらなる改善（BGM、音、SRS 回転など）は要望があれば追加します。
