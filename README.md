# シンプルオセロ (Reversi)

- 場所: `C:\workspace\othello`
- 使い方: フォルダ内の `index.html` をブラウザで開いてください。
- コマはページ読み込み時にCanvasでPNGを生成して使用します（`black`/`white` の画像は動的に生成されます）。

CPU 対戦について:
- UI の「モード」から「CPU対戦」を選択できます。
- CPU は「黒」または「白」を選べ、レベルは「やさしい（ランダム）」か「つよい（ひっくり返しが多い手を選択）」があります。

モバイルでのプレイ:
- モバイル（スマホ/タブレット）で操作しやすいように、タッチ操作向けの最適化を行いました。
- 画面が狭い場合は UI が縦に並び、ボタンや盤面のタッチ領域が大きくなります。

機能:
- 8x8 の盤面、基本的な反転ロジック、合法手表示、パス判定、スコア表示、リセットボタン。

カスタマイズ:
- `styles.css` で見た目を調整できます。
- 実際のPNGファイルを `assets/images/black.png` / `assets/images/white.png` に置いて、`script.js` を修正してそちらを読み込むことも可能です。
